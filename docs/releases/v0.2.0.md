# ClearCredit v0.2.0

## Highlights
- Admin-managed pricing catalog for one-time, monthly, and yearly plans
- Homepage and signup pricing now use the managed catalog
- Portable standalone release bundles (`.tar.gz` and `.zip`)
- Local/OpenRouter/OpenAI-compatible LLM backend support
- Portable packaged-app systemd installer and unit template
- Repo social preview image plus reusable badge set

## Release assets
- `clearcredit-0.2.0-portable.tar.gz`
- `clearcredit-0.2.0-portable.zip`
- `clearcredit-0.2.0-SHA256SUMS.txt`

## Portable quick start
```bash
tar -xzf clearcredit-0.2.0-portable.tar.gz
cd clearcredit-0.2.0
./scripts/install-portable.sh /opt/clearcredit
cd /opt/clearcredit
cp .env.example .env
./scripts/run-portable.sh
```

## Optional persistent service
```bash
cd /opt/clearcredit
sudo ./scripts/install-portable-service.sh
```
