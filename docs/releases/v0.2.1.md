## Release assets

- `clearcredit-0.2.1-portable.zip`
- `clearcredit-0.2.1-SHA256SUMS.txt`

## Portable quick start
```bash
tar -xzf clearcredit-0.2.1-portable.tar.gz
cd clearcredit-0.2.1
./scripts/install-portable.sh /opt/clearcredit
cd /opt/clearcredit
cp .env.example .env
./scripts/run-portable.sh
```

## Optional persistent service
```bash
cd /opt/clearcredit
sudo ./scripts/install-portable-service.sh
```
